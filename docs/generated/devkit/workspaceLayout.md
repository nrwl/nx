# Function: workspaceLayout

▸ **workspaceLayout**(): `Object`

Returns information about where apps and libs will be created.

#### Returns

`Object`

| Name      | Type     |
| :-------- | :------- |
| `appsDir` | `string` |
| `libsDir` | `string` |

**`Deprecated`**

Workspace Layout will be removed in Nx v18. Generators should accept the full path via a --directory argument.
