# Getting Started with Nx

Welcome to Nx! This section will help you get started with Nx and understand its core concepts.

## What is Nx?

Nx is a powerful build system with first-class monorepo support and excellent developer experience. It helps you:

- **Build faster** with advanced caching and task orchestration
- **Scale better** with code organization and boundaries
- **Ship with confidence** using automated testing and code quality tools
