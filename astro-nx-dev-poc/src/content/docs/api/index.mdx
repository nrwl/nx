# API Reference

This section contains comprehensive API documentation for all Nx tools and plugins.

## Core Tools

- [Nx CLI Reference](/api/nx-cli) - Complete reference for all Nx CLI commands
- [Nx Cloud CLI](/api/nx-cloud-cli) - Commands for Nx Cloud integration

## Plugins

Nx provides a rich ecosystem of plugins for various frameworks and tools:

- [Plugin Overview](/api/plugins) - Learn about Nx plugins
- Browse individual plugin documentation in the sidebar

Each plugin documentation includes:

- **Generators** - Code generation tools
- **Executors** - Build and task runners
- **Migrations** - Automated code updates

## Using the API Reference

The API reference provides detailed information about:

- Command syntax and options
- Configuration schemas
- Available generators and executors
- Migration paths for updates

Use the sidebar to navigate to specific plugin documentation.
