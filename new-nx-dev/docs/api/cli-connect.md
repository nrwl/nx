# `nx connect`

Connect workspace to Nx Cloud.

## Aliases

- `connect-to-nx-cloud`


## Usage

```bash
nx connect [options]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--generateToken` | boolean | No description available |  |
| `--verbose` | boolean | Enable verbose logging | `false` |
