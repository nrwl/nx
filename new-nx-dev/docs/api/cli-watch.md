# `nx watch`

Watch for changes within projects, and execute commands.



## Usage

```bash
nx watch [options]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--all` | boolean | No description available |  |
| `--command` | string | No description available |  |
| `--includeDependentProjects` | boolean | No description available (alias: `-d`) |  |
| `--includeGlobalWorkspaceFiles` | boolean | No description available (alias: `-g`) |  |
| `--projects` | string | No description available (alias: `-p`) |  |
| `--verbose` | boolean | No description available |  |
