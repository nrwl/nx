# `nx daemon`

Prints information about the Nx Daemon process or starts a daemon process.



## Usage

```bash
nx daemon [options]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--start` | boolean | No description available | `false` |
| `--stop` | boolean | No description available | `false` |
