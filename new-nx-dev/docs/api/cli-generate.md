# `nx generate`

Generate or update source code (e.g., nx generate @nx/js:lib mylib).

## Aliases

- `g`


## Usage

```bash
nx generate <generator> [_..] [options]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--dryRun` | boolean | Preview the changes without updating files. (alias: `-d`) | `false` |
| `--interactive` | boolean | When false disables interactive input prompts for options. | `true` |
| `--quiet` | boolean | Hides logs from tree operations (e.g.  |  |
| `--verbose` | boolean | Enable verbose logging | `false` |
