# `nx add`

Install a plugin and initialize it.



## Usage

```bash
nx add <packageSpecifier> [options]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--updatePackageScripts` | boolean | No description available |  |
| `--verbose` | boolean | Enable verbose logging | `false` |
