# `nx sync`

Sync the workspace files by running all the sync generators.



## Usage

```bash
nx sync [options]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--verbose` | boolean | Enable verbose logging | `false` |
