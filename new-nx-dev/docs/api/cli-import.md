# `nx import`

Import code and git history from another repository into this repository.



## Usage

```bash
nx import [sourceRepository] [destinationDirectory] [options]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--depth` | number | No description available |  |
| `--interactive` | boolean | No description available | `true` |
| `--ref` | string | No description available |  |
| `--sourceDirectory` | string | No description available (alias: `-source`) |  |
| `--verbose` | boolean | Enable verbose logging | `false` |
