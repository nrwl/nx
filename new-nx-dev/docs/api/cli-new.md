# `nx new`

The folder where the new workspace is going to be created.



## Usage

```bash
nx new [_..] [options]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--interactive` | boolean | When false disables interactive input prompts for options. | `true` |
| `--nxWorkspaceRoot` | string | The folder where the new workspace is going to be created. |  |
