# `nx reset`

Clears cached Nx artifacts and metadata about the workspace and shuts down the Nx Daemon.

## Aliases

- `clear-cache`


## Usage

```bash
nx reset [options]
```

## Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `--onlyCache` | boolean | No description available |  |
| `--onlyCloud` | boolean | No description available |  |
| `--onlyDaemon` | boolean | No description available |  |
| `--onlyWorkspaceData` | boolean | No description available |  |
