#### Update `@angular/cli` to `~19.0.0`

Update the version of the Angular CLI if it is specified in `package.json`

#### Sample Code Changes

Update in `devDependencies`:

##### Before

```json title="package.json"
{
  "devDependencies": {
    "@angular/cli": "~13.3.0"
  }
}
```

##### After

```json title="package.json"
{
  "devDependencies": {
    "@angular/cli": "~19.0.0"
  }
}
```

Update in `dependencies`:

##### Before

```json title="package.json"
{
  "dependencies": {
    "@angular/cli": "~13.3.0"
  }
}
```

##### After

```json title="package.json"
{
  "dependencies": {
    "@angular/cli": "~19.0.0"
  }
}
```
